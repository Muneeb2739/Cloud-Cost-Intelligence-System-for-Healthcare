# Cloud Cost Intelligence System for Healthcare

## Overview
This project analyzes healthcare cloud cost data to identify trends, anomalies, and seasonal patterns.

## Features
- Cost analysis by department and service
- Monthly trend visualization
- Seasonal impact analysis
- Rule-based anomaly detection
- Automated insights and recommendations

## Technologies Used
- Python
- Pandas, NumPy
- Matplotlib
- Streamlit

## How to Run
```bash
streamlit run app.py
